🎯 LANGKAH BERIKUTNYA
====================

1. Buka terminal di folder absensi-digital

2. Jalankan command:
   npm install --workspaces

3. Jalankan admin panel:
   npm run dev

4. Buka browser:
   http://localhost:3000

Catatan: Folder node_modules akan dibuat otomatis saat npm install
