📊 RINGKASAN SETUP
==================

✅ Admin Panel (Next.js + Tailwind)
   - Framework: Next.js 14
   - Styling: Tailwind CSS
   - Pages: Home, Dashboard, Employees, Attendance
   - Components: Sidebar, Card, Table

✅ Mobile App (Expo + React Native)
   - Framework: Expo
   - UI: React Native + NativeWind
   - Screens: Home, Attendance, History, Profile

✅ Shared Configuration
   - Tailwind Theme
   - TypeScript Config
   - ESLint Rules

✅ Environment Files
   - admin/.env.local (sudah dibuat)
   - mobile/.env (sudah dibuat)

✅ Root Configuration
   - package.json (npm run dev → admin panel)
   - .npmrc (npm workspaces)

🚀 CARA MENGGUNAKAN
===================

1. Buka PowerShell/CMD di folder absensi-digital

2. Jalankan:
   npm install --workspaces

3. Tunggu selesai (2-5 menit), kemudian jalankan:
   npm run dev

4. Tunggu sampai muncul:
   > ready - started server on 0.0.0.0:3000

5. Buka browser:
   http://localhost:3000

📁 FOLDER YANG AKAN DIBUAT
==========================

Saat npm install --workspaces, folder berikut akan dibuat:

✓ node_modules/          (Root dependencies)
✓ admin/node_modules/    (Admin dependencies)
✓ mobile/node_modules/   (Mobile dependencies)
